# C89 Explained (Unofficial)

This repository contains **original educational explanations**
of the C programming language as defined in ANSI C (C89).

⚠️ This project is **NOT a translation, reproduction, or derivative work**
of the official ANSI or ISO C standard documents.

All text is written in original wording for learning and teaching purposes.
No official standard text is reproduced or translated.
