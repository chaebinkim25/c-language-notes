## 3.1 C 코드의 글자들에 의미를 부여하는 단위

영어 에세이나 C 코드나 알파벳과 문장 부호를 모아놓은 것이다. 

우리가 영어 문장을 읽을 때, 왼쪽 끝에 띄어쓰기 ` `가 있는 ` apple?`라는 문자들을 보면 ` `, `a`, `p`, `p`, `l`, `e`, `?`로 

따로 떼어 생각하지 않고, 띄어쓰기를 제외한 알파벳들을 묶어서 `apple`이라는 하나의 단어와 `?`라는 문장부호로 인식한다.

C 코드도 정해진 규칙에 따라서 붙어 있는 문자들끼리 하나의 의미 있는 단위를 이룬다. 

프로그래밍 용어로 이를 토큰(token)이라고 부른다.
