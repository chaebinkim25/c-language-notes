## 3.1 C 코드의 글자들에 의미를 부여하는 단위

한글 에세이나 C 코드나 글자들과 문장 부호를 모아놓은 것이다. 

우리가 한글 문장을 읽을 때, `코딩`이라는 문자들을 보면 `코`와 `딩`으로 

따로 떼어 생각하지 않고, 띄어쓰기를 제외한 글자들을 묶어서 `코딩`이라는 하나의 단어로 인식한다.

C 코드도 정해진 규칙에 따라서 붙어 있는 문자들끼리 하나의 의미 있는 단위를 이룬다. 

프로그래밍 용어로 이를 토큰(token)이라고 부른다.

## 3.2 C 코드를 토큰으로 나누는 규칙

`코드를 코딩하다?`라는 문장을 토큰으로 나눠보면 `코드를`, `코딩하다`, `?`가 된다. 

토큰으로 나눌때 쓴 규칙은 

- 띄어쓰기는 토큰을 분리한다
- 문장부호는 한글과 별개로 하나의 토큰이 된다

였다.

C 코드를 토큰으로 나눌 때도 같은 규칙이 적용된다. 다음 C 코드 

```c
int main()
{
}
```

을 토큰으로 나누면, 

띄어쓰기가 토큰을 분리한다는 규칙을 적용해서 `int` 토큰이 나오고, 

문장부호는 한글과 별개로 하나의 토큰이 된다는 규칙을 적용해서 `main`, `(`, `)`, `{`, `}` 토큰이 나온다.
